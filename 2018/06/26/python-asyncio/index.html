note

asyncio 是在 Python 3.4 中引入的异步 IO 标准库.

简单介绍一下什么是同步 IO / 异步 IO.

async/await/future

Python 3.5 引出的语法糖

async 创建协程, await 当协程阻塞时将其挂起并让步操作给其它协程, future 包装协程.

通过 asyncio.sleep() 模拟 IO 操作.
