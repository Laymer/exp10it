powershell execution bypass

bypass

PowerShell execution policy 是用来决定哪些类型的 PowerShell 代码可以在系统中运行.

用 Get-ExecutionPolicy 查看当前的策略.

Bypass

将 ExecutionPolicy 设置为 Bypass 绕过策略限制.

UnRestricted

将 ExecutionPolicy 设置为 UnRestricted 更改策略.

RemoteSigned

将 ExecutionPolicy 设置为 RemoteSigned 更改策略.

Pipe

通过管道执行脚本.

类似的还有 type 命令.

IEX

通过 IEX 远程下载脚本执行 不受策略限制.

Encode

unicode + base64 编码执行.

Command

好像会有引号的转义问题, 更改了下执行的脚本.

Invoke-Expression

Invoke-Expression 将它接受到的任何字符串当作 powershell 代码执行.
