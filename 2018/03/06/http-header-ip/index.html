http 头伪造 IP

note

http 头 伪造 ip

大多都跟反向代理有关

X-Originating-IP 主要出现在邮件头里

X-Forwarded-For 不多说 代理转发的 IP

X-Real-IP 跟 X-Forwarded-For 类似

X-Remote-IP 和 X-Remote-Addr 远程主机的 IP

因为都是从脚本层获取的 所以可以进行伪造
