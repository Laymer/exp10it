mysql load data local

note

当 LOAD DATA INFILE 指定 LOCAL 时, 用户无需 FILE 权限, 文件将由客户端读取, 并把数据发送至服务器上.

以当前执行命令用户的权限读取文件.

例如从 phpmyadmin 中执行语句, 用户权限就为 web 容器的权限.

如果显示不全或者其它奇怪的原因就在语句后面加上 LINES TERMINATED BY '\0'.
