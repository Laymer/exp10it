有的命令含有 /domain, 多数指在域控上查询, 即在域控上执行该命令, 并回显结果

其中 dsquery 命令仅能在域控上使用

net view 获取域/工作组内计算机 (可能不全)

net view /domain  获取所有域/工作组

net config Workstation 获取当前计算机信息

nbtstat -A 192.168.1.100 获取 IP 对应 MAC 地址

ping -n 1 DC -4 根据计算机名获取对应 IP, -4 指获取 IPv4 地址

dsquery user 获取域内所有用户, 以 ldap 方式回显

dsquery computer 获取域内所有计算机, 以 ldap 方式回显

net user /domain 获取域内所有用户

net localgroup /domain 获取域内所有本地组

net group /domain 获取域内所有全局组

其中部分域组如下, 其中域控独立于 Domain Computers 组, 故仅能在 Domain Controllers 组中被查询到

Domain Admins 域内管理员组

Domain Computers 域内计算机组

Domain Controllers 域内控制器组

Domain Guests 域内来宾用户组

Domain Users 域内用户组

获取域组内成员

net group "Domain Controllers" /domain

net group "Domain Admins" /domain

创建域用户/提升为域管理员

net user exp10it 1234abcd!@#$ /add /domain

net group "Domain Adminis" exp10it /add /domain

设置共享, 其中默认共享为 盘符$, 如 C 盘共享为 c$

net view \\DC 查询共享

net use z: \\DC\c$ 匿名映射盘符

net use z: \\DC\c$ password /user:username 登录映射盘符

net use z: /del 删除映射

net share name$=d:\web 创建共享

net use z: \\DC\name$ 映射共享

net use z: \\DC\name$ password /user:username 登录映射共享

net share name$ /del 删除共享

定位域控

net time /domain

nslookup -type=SRV _ldap._tcp.

ipconfig
